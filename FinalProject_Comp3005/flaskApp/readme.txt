Final Project: Art Database

The database file is located in flaskApp/data/artwork.db
The source files for this project are app.py, dbManager.py, artwork.db, and initialiseDb.sql

The dependencies for this program include, flask, and sqlite3.
To run the program from the flaskApp directory type "flask run" into the command line, from there the app should open in your browser.
The database can be accessed separately, an optional initialiseDb.sql file as been included although is not required to use the database.
